# プライバシーポリシー

最終更新日: 2026年5月29日

本プライバシーポリシーは、「WondayWall」（以下「本アプリ」といいます）の利用に際し、ユーザーの情報およびデータの取り扱いについて定めるものです。

## はじめに

本アプリは、ユーザーの予定、RSSニュース、興味キーワード、追加指示をもとに壁紙画像を生成する Windows / iOS / Android 向けアプリです。
本アプリの提供者は、通常の利用においてユーザーの個人情報、予定内容、設定内容、生成画像、生成履歴を収集しません。

ただし、本アプリは機能提供のため、端末上で保存した情報を読み書きし、Google AI / Gemini、Google Calendar、RSS配信元、記事ページ、OGP画像取得先などの外部サービスへ必要な通信を行います。
また、各OSのカレンダー、写真、通知、壁紙設定などの機能を、ユーザーの許可または設定に基づいて利用します。

## OS別に保存される情報

### Windows

Windows版は、次の情報をユーザーのPC上に保存します。

- 設定ファイル: `%LocalAppData%\StudioFreesia\WondayWall\config.json`
- 生成履歴: `%LocalAppData%\StudioFreesia\WondayWall\history.json`
- 生成画像: `%LocalAppData%\StudioFreesia\WondayWall\wallpapers\`
- Google Calendar OAuth トークン: `%LocalAppData%\StudioFreesia\WondayWall\calendar-token\`
- Google AI API キー: Windows 資格情報

Google AI API キーは Windows 資格情報に保存され、設定ファイルには直接保存されません。

### iOS

iOS版は、次の情報をユーザーの端末上に保存します。

- アプリ内設定
- 生成履歴
- 生成画像
- 取得対象カレンダーの識別子
- Google AI API キー

Google AI API キーは Keychain に保存されます。
生成画像はアプリ領域に保存され、写真ライブラリ保存を利用した場合は、写真ライブラリの WondayWall アルバムにも保存されます。

### Android

Android版は、次の情報をユーザーの端末上に保存します。

- アプリ設定
- 生成履歴
- 生成画像
- 取得対象カレンダーの識別子
- Google AI API キー

アプリ設定は DataStore に保存されます。
Google AI API キーは Tink と Android Keystore を使って暗号化保存され、DataStore には平文保存されません。
生成画像はアプリ領域に保存され、ユーザー操作により写真/ギャラリーへ保存される場合があります。

## OS別の権限とカレンダー取得

### Windows

Windows版は、ユーザーがGoogle Calendar連携を有効にした場合、Google Calendar API の読み取り専用スコープを使用して予定情報を取得します。
取得する予定情報には、予定のタイトル、開始時刻、終了時刻、場所、説明が含まれる場合があります。
Google Calendar OAuth トークンはユーザーのPC上に保存され、本アプリ提供者は取得、保存、閲覧、共有しません。

### iOS

iOS版は、ユーザーが許可した場合、iOSのカレンダー機能を通じて端末上のカレンダー予定を取得します。
取得する予定情報には、予定のタイトル、開始時刻、終了時刻、場所、説明が含まれる場合があります。
iOS版は、通常アプリからホーム画面・ロック画面の壁紙を直接変更しません。
写真ライブラリ保存、共有、通知、壁紙設定手順の表示を利用する場合、それぞれOSの許可またはユーザー操作が必要です。

### Android

Android版は、ユーザーが `READ_CALENDAR` 権限を許可した場合、Calendar Provider / `CalendarContract` を通じて端末に同期済みのカレンダー予定を取得します。
初期版では、Android版は Google Calendar API や Google OAuth による予定取得を行いません。
取得する予定情報には、予定のタイトル、開始時刻、終了時刻、場所、説明が含まれる場合があります。

Android版は、`WallpaperManager` によりホーム画面の壁紙を適用します。
ユーザー設定により、ロック画面にも追加で適用する場合があります。
通知、写真/ギャラリー保存、壁紙設定に関する権限やOS機能は、ユーザーの許可または操作に基づいて利用します。

## RSS、記事ページ、OGP画像の取り扱い

本アプリは、ユーザーが登録したRSSフィードURLへアクセスし、記事タイトル、概要、記事URL、公開日時を取得します。
また、壁紙生成に利用する記事について、記事ページのOGP情報やOGP画像を取得する場合があります。

RSS配信元、記事ページ、画像配信元は、それぞれのサービスまたはWebサイトのポリシーに基づいて、アクセスログ、IPアドレス、ユーザーエージェントなどを記録する場合があります。

## Google AI / Gemini への送信情報

本アプリは、壁紙生成のためにGoogle AI / Gemini APIを利用します。
APIリクエストには、次の情報が含まれる場合があります。

- ユーザーが入力した興味キーワードや追加指示
- Windows版で Google Calendar API から取得した予定情報
- iOS版で iOS カレンダーから取得した予定情報
- Android版で Calendar Provider / `CalendarContract` から取得した予定情報
- RSSフィードや記事ページから取得したニュース情報
- 参照画像として利用するOGP画像
- 現在の壁紙または過去に生成した壁紙画像をベースに生成する設定が有効な場合、その画像

これらの情報は、ユーザーが設定したGoogle AI APIキーを使用してGoogle AI / Geminiへ送信されます。
本アプリ提供者は、送信内容や生成結果を収集しません。
Google AI / Geminiに送信された情報の取り扱いは、Googleの規約およびプライバシーポリシーに従います。

## 生成画像と壁紙適用

### Windows

Windows版は、生成画像をデスクトップ壁紙として適用します。
ユーザーがロック画面更新を有効にしている場合は、生成画像をロック画面にも適用します。

### iOS

iOS版は、生成画像をアプリ領域に保存します。
ユーザーが写真ライブラリ保存を選択した場合、生成画像を写真ライブラリの WondayWall アルバムに保存します。
共有シートや壁紙設定手順の表示により、ユーザーが手動で壁紙に設定できる状態にします。
本アプリは、iOSのホーム画面・ロック画面壁紙を直接変更しません。

### Android

Android版は、生成画像をアプリ領域に保存し、`WallpaperManager` によりホーム画面の壁紙として適用します。
ユーザーがロック画面更新を有効にしている場合は、ロック画面にも追加で適用する場合があります。
ユーザー操作により、生成画像を写真/ギャラリーへ保存したり、共有したりできます。

## データの共有

本アプリ提供者は、ユーザーの個人情報、予定情報、RSS設定、APIキー、OAuthトークン、端末内データ、生成画像、生成履歴を第三者へ販売、共有、転送しません。

ただし、本アプリの機能上、ユーザーの操作や設定に基づいて、Google、RSS配信元、記事ページ、画像配信元などの外部サービスへ通信が行われます。
外部サービスでのデータ取り扱いは、それぞれのサービスの規約およびプライバシーポリシーに従います。

## データの削除方法

### Windows

本アプリが保存した設定、履歴、生成画像、Google Calendar OAuthトークンを削除するには、次のフォルダを削除してください。

`%LocalAppData%\StudioFreesia\WondayWall\`

Google AI APIキーを削除するには、Windows 資格情報から `StudioFreesia.WondayWall.AppConfig` に関連する資格情報を削除してください。

### iOS

アプリ領域に保存された設定、履歴、生成画像は、アプリ内の設定変更またはアプリの削除により削除できます。
写真ライブラリに保存した画像は、写真アプリから削除してください。
カレンダー、写真、通知の許可は iOS の設定アプリから変更できます。

### Android

アプリ領域に保存された設定、履歴、生成画像は、Android のアプリ情報画面からアプリデータを削除するか、アプリをアンインストールすることで削除できます。
写真/ギャラリーに保存した画像は、端末の写真またはギャラリーアプリから削除してください。
カレンダー、通知、壁紙関連の許可は Android の設定アプリから変更できます。
