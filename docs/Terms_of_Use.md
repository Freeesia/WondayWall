# WondayWall 利用規約

最終更新日: 2026年5月29日

本利用規約（以下「本規約」）は、Freeesia（以下「当方」）が提供する WondayWall（以下「本アプリ」）の利用条件を定めます。
本アプリをインストールまたは利用した場合、本規約に同意したものとみなします。
ソースコードの利用、改変、再配布には別途 [MIT License](https://github.com/Freeesia/WondayWall/blob/main/LICENSE) が適用されます。

## 1. 本アプリの用途

本アプリは、Windows / iOS / Android 向けに、ユーザーのカレンダー予定、登録した RSS フィード、追加プロンプト等をもとに、Google Gemini API を使用して壁紙画像を生成するツールです。
生成した画像の壁紙への適用、保存、共有、壁紙設定支援は、各OSが許可する範囲で行います。
生成結果の正確性、品質、適合性を保証しません。

## 2. 外部サービスとOS機能

本アプリは、次の外部サービスと連携する場合があります。

- Google Gemini / Google AI
- Google Calendar API（Windows版のカレンダー連携）
- RSS 配信元
- 記事ページ
- OGP 画像取得先

本アプリは、次のOS機能を利用する場合があります。

- Windows のデスクトップ壁紙、ロック画面、Task Scheduler、Windows 資格情報
- iOS のカレンダー、写真、共有、通知、Keychain、バックグラウンドタスク
- Android の Calendar Provider / `CalendarContract`、`WallpaperManager`、通知、WorkManager、Android Keystore、写真/ギャラリー保存

利用者は、各外部サービスおよび各OS機能の利用規約、プライバシーポリシー、料金、API制限、権限設定を確認し、自己の責任で利用するものとします。
第三者サービスまたはOS機能の変更、停止、課金、制限により本アプリが利用できなくなる場合があります。
