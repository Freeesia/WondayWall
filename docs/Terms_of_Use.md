## 1. 本アプリの用途

本アプリは、ユーザーの Google カレンダー、登録した RSS フィード、追加プロンプト等をもとに、Google Gemini API を使用して壁紙画像を生成し、Windows のデスクトップ壁紙等へ適用するツールです。
生成結果の正確性、品質、適合性を保証しません。

利用者は各サービスの利用規約、プライバシーポリシー、料金、API 制限を確認し、自己の責任で利用するものとします。
第三者サービスの変更、停止、課金、制限により本アプリが利用できなくなる場合があります。

Google AI API キー、カレンダー認証トークン、設定、履歴、生成画像は利用者の Windows 環境の `%LocalAppData%\WondayWall\` 以下に保存されます。
利用者は端末、アカウント、認証情報、生成画像を適切に管理してください。
